Description

GeekStuff company introduces their eCommerce project with the launch of a multi-page website that allows visitors to learn about crafting, pledge support for the development of crafting skills, buy crafting products, ask questions about the website and products by contacting the staff, and share the info with social media platforms.

Once the product is ready to ship, orders will be taken via a secure PayPal payment gateway connection or can be pay with cash on delivery. The new domain name will point to a hosted server with a monthly plan for malware prevention.

Content will be managed using WordPress plugins, such as using WooCommerce for the payment methods and payment getaway. Sucuri security, and WP Security as firewalls to make the website IP unique so it can only be accessed by authorized users, and protect the website from hacker attacks. UpdraftPlus for automatic backup of the website to a Google drive or email. Other optimizer plugins like ShortPixel Image Optimizer and LiteSpeed Cache.

Marketing staff will each have their own login credentials to add and update text and media. Two administrators will have login credentials for the hosting, domain, and website accounts.

At least one accountant will have access to the host account to ensure automatic hosting and domain payments are continued and updated. At least one accountant will set up the PayPal WooCommerce (and/or other payment gateway)once the site begins taking orders. Success and maintenance of the website will be monitored using WooCommerce purchases, Google Search Console, Google Analytics, and social media shares.


Backup: the website auto backups every day with the WordPress plugin UpdraftPlus. This backup will be sent to an email, Google drive or cloud service.

Other options:

  1. VaultPress powered by Jetpack. This plugin can backup automatically your WordPress files to Google drive, email and cloud services.
  2. BackupBuddy one of the most popular backup plugins in WordPress that can backup your files in Dropbox, Amazon S3 and other clouds. It is not free.
  3. BackWPUp is a free plugin that can store backups in Dropbox, send to email and or to your local computer.
  4. BackUpWordPress is a backup plugin that allows users to auto backup their files to Google drive, FTP, Dropbox and other clouds.
